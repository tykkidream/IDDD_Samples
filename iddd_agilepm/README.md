敏捷项目管理上下文（Agile Project Management Context）
=================================================

需求说明
------------

这是一套敏捷项目管理软件，使用Scrum作为项目管理方式，并且采用增量式的管理框架。

该产品翻用传统的Scrum项目管理模型，功能包括：

- 产品（product）
- 产品负责人（product owner）
- 团队（team）
- 待定项（backlog item）
- 计划发布（planned release）
- 冲刺（sprint）
